# How Much A Heatwave Cost
### Visualizing the Impact of Heatwaves and Drought in Kansas
***
Despite being known for its tornadoes, the extreme weather event that causes the most fatalities and results in the largest economic loss is extreme heat. The goal of this repository is to create visualizations that help communicate this often ignored issue within the state using data on billion-dollar climate disasters within the United States.
